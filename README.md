
# Flask

Requests feito a partir de um arquivo JSON, feito na entrada do usuário.




## Instalação


```bash
  pip install flask
```

## Rodar aplicação

```bash
$ export FLASK_APP=app
$ flask run
 * Running on http://127.0.0.1:5000/
```
